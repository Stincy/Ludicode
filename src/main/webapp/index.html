Descpription

Ludicode est un site web permettant à n'importe qui d'apprendre et de comprendre l'algorithmie et ce qui se cache derriere les jeux et les programmes. Vous pouvez ainsi vous inscrire, vous connecter et acceder aux différents niveaux. Vous pouvez aussi acceder aux classements ( Leaderboard ) de tous les joueurs et de tous les niveaux.
